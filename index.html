About me

Hello! Im Lucas Acuña, a junior developer from Uruguay who has a big interest in the IT world. I started my learning about the middle of 2022 with a web development course in which i learn the basics of html, css, php, mysql and a bit of javascript. Now im coursing "Jovenes a Programar"; a intensive course of development to improve my web programming tools and learn about testing. Also im working/studying in a big tech enterprise as junior trainee doing udemy enterprise courses, learning and practising salesforce with trailhead project badges and a lot of another stuff.



Tech stack


My bases are Html and Css with frameworks and libraries like bootstrap or materializecss, also i learned basic data base management using php and mysql in a course.
Currently, I am focused on improving my skills and learnings using Udemy business courses while working on programming and also learning apex, javascript and soql for salesforce.
